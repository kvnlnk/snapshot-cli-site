$ features

>_

Pure SVG Output

Zero browser dependencies. No Puppeteer, no Playwright, no headless Chrome. Every screenshot is clean, standalone SVG.

>_

ANSI Color Parsing

Pipe any colored terminal output through snapshot. Full support for 8/16/256 ANSI color codes and SGR parameters.

>_

5 Built-in Themes

Retro-green, light, minimal, nord, and default. Each theme controls background, foreground, chrome, and accent colors.

>_

Flexible Input

Read from stdin pipes, pass a file path, or provide a raw string. Works in any shell pipeline.

>_

Command Execution

Run a command and capture its output in one step. Perfect for documenting CLI tools in READMEs.

>_

Lightweight

Single binary, no runtime dependencies. Installs via npx or direct download in seconds.
